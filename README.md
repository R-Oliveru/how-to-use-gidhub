# how-to-use-gidhub
记录了新手学习如何使用github的心得体会

一、我的核心需求
版本管理与协作（核心需求）

代码版本管理：记录每次修改，方便回退和查看历史。
多人协作开发：与同伴高效同步代码，避免冲突，管理不同成员的贡献。
代码学习与复用（高效学习）

寻找现成项目：在 GitHub 上搜索特定技术（如小程序、React Native、Flutter）的优质开源项目。
学习与分析：下载（克隆）项目到本地，在 CodeBuddy 中打开，运行、阅读和修改代码，理解其架构与实现。
